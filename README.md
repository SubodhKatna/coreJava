## ğŸ’» How to Run This Project

This project is Eclipse-ready. Follow these steps to set it up locally:

### Prerequisites
* Ensure **Java Development Kit (JDK)** is installed on your machine.
* Ensure **Eclipse IDE** is installed.

### Installation Steps

1. **Clone the Repository**  
   Open your terminal or Git Bash and run:

   ```bash
   git clone https://github.com/SubodhKatna/Java.git

2.  **Import into Eclipse**
    * Open Eclipse.
    * Go to `File` > `Open Projects from File System...`
    * Click `Directory` and browse to the cloned `coreJava` folder.
    * Click `Finish`.

3.  **Run a Module**
    * Expand the project explorer.
    * Navigate to any package (e.g., `basics`).
    * Right-click on a Java file (e.g., `HelloWorld.java`).
    * Select **Run As** > **Java Application**.
